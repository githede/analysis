# [데이터 수집 계획 및 보고서] <br> 🌟효과적인 마케팅을 위한 온라인 쇼핑 소비자 평가 데이터 분석

## ⏰작업 기간
2023.05.16.-2023.05.26
## 💡데이터 수집 목적
지속적으로 온라인을 통한 소비자거래가 늘어남에 따라 2019년부터 2021년도까지의 3년간 서울시 온라인 쇼핑을 이용한 소비자를 대상으로 집계된 평가데이터를 파악하고, 향후 자사 고객층에게 효과적인 마케팅 기획을 위함
